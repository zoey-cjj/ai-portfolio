📘 Case 1：智能问答系统（RAG）

1️⃣ 项目背景

客服人力占运营成本 35%，70% 问题重复。
目标：自动解决率 ≥ 70%，幻觉率 ≤ 5%。

2️⃣ 为什么必须用 AI？

规则引擎无法覆盖复杂语义；人工不可扩展。

3️⃣ 方案设计

用户输入 → 意图识别 → 检索(RAG) → LLM 生成 → 安全过滤 → 输出

4️⃣ 评测体系

  • 准确性 / 完整性 / 幻觉 / 合规
  • Golden Set：20 条典型问题

📘 Case 2：内容生成 Copilot

1️⃣ 背景

运营内容生产周期长，人工成本高。

2️⃣ 产品方案

Prompt + 模板 + 人工审核闭环

3️⃣ 成功指标

生产效率提升 ≥ 40%，人工修改率 ≤ 20%

🧪 评测体系（Eval）

分数 标准
5 完全正确，无幻觉
3 部分正确
1 错误 / 有害
